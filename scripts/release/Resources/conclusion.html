Interceptor is installed. Two quick steps left.

1Load the browser extension

Chrome and Brave do not allow installers to load extensions automatically — you have to do this once, by hand:

  1. Open chrome://extensions/ (or brave://extensions/)
  2. Toggle Developer mode on (top right)
  3. Click Load unpacked
  4. Select this folder:
    /Library/Application Support/Interceptor/extension

2Allow Privacy & Security prompts on first macOS command

The first time you run an interceptor macos command, macOS will prompt for Accessibility, Screen Recording, and Apple Events for interceptor-bridge. Allow each one in System Settings → Privacy & Security.

Try it

Open a terminal and run:

interceptor open "https://example.com"

To uninstall later, run
sudo bash "/Library/Application Support/Interceptor/uninstall.sh"
