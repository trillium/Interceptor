Interceptor (Full) is installed. Three quick steps left.

1Load or reload the browser extension

Chrome and Brave do not allow installers to load or refresh extensions automatically — do this once by hand, and after package updates:

  1. Open chrome://extensions/ (or brave://extensions/)
  2. Toggle Developer mode on (top right)
  3. Click Load unpacked
  4. Select this folder:
    /Library/Application Support/Interceptor/extension
  5. If Interceptor was already loaded from this folder, click its Reload button instead.

After the extension is connected, interceptor reload also refreshes the running extension service worker.

2Allow Privacy & Security prompts on first macOS command

The first time you run an interceptor macos command, macOS will prompt for Accessibility, Screen Recording, and Apple Events for interceptor-bridge. Allow each one in System Settings → Privacy & Security.

3Optional: register skill packs with your AI tooling

Interceptor ships skill packs that teach AI agents how to drive both the browser and macOS surfaces. Find your runtime below and paste its block into a terminal. Skip the runtimes you don’t use. Existing symlinks of the same name are replaced safely; nothing else is touched.

mkdir -p ~/.claude/skills
ln -sfn "/Library/Application Support/Interceptor/skills/interceptor" ~/.claude/skills/interceptor

Try it

Open a terminal and run:

interceptor open "https://example.com"

Or, after granting Privacy & Security:

interceptor macos tree

To uninstall later, run
sudo bash "/Library/Application Support/Interceptor/uninstall.sh"
To downgrade Full → Browser only:
sudo bash "/Library/Application Support/Interceptor/uninstall.sh" --bridge-only
